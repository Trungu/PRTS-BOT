To install.
create a virtual env
```bash
python3 -m venv .venv
```
install requirements
```bash
pip install -r requirements.txt
```
copy .env.example to .env
```bash
cp .env.example .env
```
edit your data in .env
run the bot
```bash
python main.py
```
